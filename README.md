# -Entrega-4-Din-mica-Molecular-amb-SIESTA
arxius de entrega de la pràctica 4: dinamica molecular amb siesta

El arxiu report prac MD.pdf es tracta del arxiu pdf amb el informe de la pràctica.

Dins la carpeta .zip prac molecular dynamics.zip es troben TOTS els arxius obtinguts de la simulació de SIESTA en dos carpetes.

La carpeta OldPrac5 conté tots els arxius de la simulació original feta a classe.

La carpeta DoblePrac5 conté tots els arxius de la simulació modificada, on els atoms d'hidrogen tenen el doble de velocitat.
